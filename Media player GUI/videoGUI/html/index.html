Introduction

Basic application that plays a video file of .avi format in the GUI This program demonstrates the use of :

  • Qt main window provides a framework for building an application's user interface
  • QMedia player class allows playing of a media source
  • The QMediaPlayer class is a high level media playback class. It can be used to playback such content as songs, movies and internet radio.


Working:

  • User can select which videos they want to play fromt the sample folder
  • Video selected is played in the main window provided
  • Video can be paused or stopped
  • Slider can be used to fast forward or backward the video
  • Corresponding progress of the video is hsown in the progress bar
